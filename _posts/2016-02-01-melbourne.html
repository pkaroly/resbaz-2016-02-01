Please sign up here - we'll be in touch! classes: - class: Main Streams tools: - name: MATLAB Data Analysis id: mat blurb: MATLAB is a powerful and versatile tool for data acquisition, analysis and visualisation. In this workshop, you’ll learn the basics of MATLAB for quick and effective data analysis. It covers all steps from reading in data, writing scripts and functions for basic analyses, and creating figures that can be shared and published. - name: R Data Analysis id: r blurb: R is widely used across a range of disciplines for statistical computing and generating publication-quality figures. In this Software Carpentry workshop, you’ll learn the fundamentals of R for data science, Unix Shell for interacting with other programs, as well as version control and useful concepts for scientific computing such as breaking down problems into modular units, reproducibility, tracking changes to code, collaborating with others, and working with structured data. - name: Python (via iPython) Data Analysis id: py blurb: Python is a powerful, general purpose programming language that has several popular scientific libraries that make it ideal for research. This workshop will equip you with the essential skills you need to use the Python programming language. It covers fundamentals as well as creating reusable functions, graphing with matplotlib and what to do to find and fix errors in your code. - name: NLTK Natural Language Processing id: nltk blurb: Researchers are often confronted with natural language datasets so large that not even teams of researchers would be able to analyse by hand. (Even just reading through the 100 million word British National Corpus, for example, would take around 10,000 hours.) Python's Natural Language Toolkit (NLTK) provides a powerful means of investigating large bodies of text for patterns that may otherwise be very difficult to find. In this workshop, you’ll learn key concepts and practices in Corpus Linguistics/Distant Reading, investigate example datasets and strategies for implementing NLTK within your own research. - name: AutoDesk 3D Printing id: ad3d blurb: Inventor 3D CAD software offers researchers an easy-to-use set of tools for 3D object design, documentation and product simulation. Digital prototyping with Inventor can help researchers design and validate physical concepts before they’re built. Turning ideas to reality with better research outcomes at reduced cost and lead-times. The training aims to teach basic principles in designing parts, assemblies, and drawings. - name: 3D Slicer 3D Printing id: 3ds blurb: 3D Slicer (Slicer) is a free and open source software package for medical image analysis and scientific visualization. Slicer’s platform provides functionalities for segmentation, registration and three-dimensional visualization of multimodal image data, as well as advanced image analysis algorithms for diffusion tensor imaging, functional magnetic resonance imaging and image-guided radiation therapy. The training aims to teach basic principles in image processing, as well as techniques in extracting 3D models for bio-simulations and 3D printing. - name: HTML, CSS, and JavaScript Web Development id: web blurb: HTML (the Hypertext Markup Language), CSS (Cascading Style Sheets), and JavaScript are the core tools for building web pages. In this workshop you'll learn the basics of structuring a website, and controlling how it looks and acts. - name: D3 Data Visualization id: d3 blurb: D3 is a JavaScript library for working with data and displaying it online. D3 helps you bring data to life using HTML, SVG, and CSS. This workshop will introduce you to the basics of using D3 to create beautiful, animated and interactive graphics out of your data. (This workshop requires beginner level HTML and CSS skills, plus experience in at least one programming language) - name: CartoDB and TileMill (MapBox) Research Maps id: map blurb: CartoDB is a simple yet powerful tool that lets you visualise data onto maps. You’ll learn how to format data to make a map visualisation. We’ll explore the range of inbuilt features that CartoDB offers and how to start customising these tools to produce a map that communicates your research data through a narrative.
TileMill lets you create customised maps to display your research. In this workshop, you’ll learn about the art of cartography and the types of data that you can access and use in TileMill. We’ll introduce you to CartoCSS, so you can customise the appearance of your map and control the level of detail that appears and create a map that you’ll be proud to publish. CartoDB is recommended but not required for attending this workshop. - class: Minor Electives tools: - name: Data Collection id: daco blurb: Electives include -
